# # Cross-Sector Stock Analysis: Risk, Return & Diversification

**Description:**  
Educational project in financial analysis: applying volatility, beta, Sharpe ratio and CAPM to real market data.

---

## Project Overview

This project demonstrates a practical application of financial analysis techniques using real stock market data.

The main goal was to analyze stocks from different sectors and evaluate their:
- risk
- return
- diversification potential

---

## Project Workflow

The work was carried out as follows:

1. A Python script was used to download and process stock market data.
2. A selection of stocks from different sectors was made to test diversification hypotheses.
3. Key financial metrics were calculated:
   - Volatility
   - Beta
   - Sharpe Ratio
   - CAPM expected return
4. Results were exported into an Excel file.
5. A final analytical report was prepared with conclusions.

---

## Project Structure

- `src/stock_analysis.py` — main script for data collection and calculations  
- `data/stock_analysis.xlsx` — generated dataset with calculated metrics  
- `report/Stock_Analysis_Report.pdf` — final report with insights and conclusions  

---

## Key Metrics Explained

- **Volatility** — measures risk (price variability)
- **Beta** — sensitivity to market movements
- **Sharpe Ratio** — risk-adjusted return
- **CAPM** — expected return based on systematic risk

---

## Results & Insights

The analysis shows how:
- stocks from different sectors behave differently in terms of risk and return
- diversification can reduce portfolio risk
- high returns are often associated with higher volatility

Detailed conclusions can be found in the report.
